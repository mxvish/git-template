# git-template
git command template

Execute each command from the top one.

```sh
git init .
```
```sh
git add $FILE
```
```sh
git commit -m "first commit"
```
```sh
git branch -M main
```
```sh
git remote add origin https://github.com/$PATH.git
```
```sh
git pull origin main
```
```sh
git push -u origin main
```
